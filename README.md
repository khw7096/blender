### Blenderset
This Code is lazypic blender setting code.

#### install
```
cd ~ && git clone https://github.com/lazypic/blender.git
```

#### how to run
- macOS
```bash
$ /Applications/Blender/blender.app/Contents/MacOS/blender --python ~/blender/init.py
```
