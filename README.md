### Blender Setting
This Repository is Lazypic Blender setting.

#### Install Env
```bash
$ cd ~ && git clone https://github.com/lazypic/blender.git
```

#### Install Path

- 블랜더는 [Blender.org](https://blender.org) 에서 다운받아주세요.
- 프로젝트마다 사용하는 블랜더 버전이 다릅니다. 설치 경로는 아래와 같습니다.
    - blender2.79: $HOME/app/blender2.79
    - blender2.81: $HOME/app/blender2.81
    - blender2.82: $HOME/app/blender2.82
    - blender2.83: $HOME/app/blender2.83


#### How to run blender with python script

```bash
$ ~/app/blender2.xx/blender --python ~/blender/init.py
```
